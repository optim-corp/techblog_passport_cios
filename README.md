## 実行手順

`app.js` 記載の ClientID及びClientSecretに各自入力してください。

```sh
cd myapp
npm i
npm run start
```

## 構築手順

```sh
express myapp
cd myapp
npm i
npm i --save optim-corp/passport-cios passport express-session
npm run start
```
